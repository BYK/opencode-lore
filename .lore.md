## Long-term Knowledge

### Gotcha

<!-- lore:019e18ec-e328-76c4-9c3c-09dbe8d51c6c -->
* **Bun tests leak into production DB when run outside repo root**: Running \`bun test\` outside the repo means \`bunfig.toml\` is not found, preload never fires, and tests silently open the production DB. Fix (now merged to main): \`db()\` throws immediately if \`NODE\_ENV=test\` and \`LORE\_DB\_PATH\` is not set to a temp path — runtime guard rather than relying on preload. \`bunfig.toml\`-based isolation is fragile due to CWD dependency.

<!-- lore:019e1919-9389-7fdc-bb21-5d353df70006 -->
* **Gateway double-compaction bug after structural compaction interception**: Two compaction bugs fixed in gateway: 1. \*\*Double-compaction\*\*: After intercepting structural compaction, \`handleCompaction\` must set \`sessionState.messageCount = currCount\`. Without this, the next autocontinue turn re-triggers structural detection. 2. \*\*Subagent compaction leak\*\*: Gateway must NOT intercept compaction for subagent sessions (detected via \`x-parent-session-id\` header). When a subagent is compacted, the gateway response becomes the subagent's \`task\_result\` returned to the parent — causing the parent to receive the lore template output and stop. Fix: check for subagent header early in compaction detection (before \`handleConversationTurn\`) and bypass interception, passing through to upstream API.
