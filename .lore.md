### Decision

### Gotcha

<!-- lore:019dfa62-9c29-746f-ab60-313983894131 -->
* **Anthropic cache TTL is 5 minutes, not 1 hour â€” inform refresh policy**: Anthropic prompt cache TTL is ~5 minutes, not 1 hour. Any prefix refresh within a warm window pays full cache write cost (~$3.75/MTok Ă— context\_size, ~$1.88 per bust for 500K). Runtime monitoring needed: track bust rate per session; alert stderr when rate > 50% after 20+ calls to catch regressing code paths early.
